# Explore Workflow

Full 8-lens deep analysis.

## Trigger
User says "analyze deeply", "what am I missing", "thorough analysis"

## Steps

1. **Understand** — Summarize what's being analyzed
2. **Sweep** — Apply each of the 8 lenses, noting findings
3. **Highlight** — Mark the most surprising or important discoveries
4. **Synthesize** — What's the overall picture? What was hidden?
5. **Recommend** — Prioritized list of things to address

## Output Format

```
# 🔍 Deep Analysis: [Topic]

## Summary
[What we're analyzing]

## Lens Findings

### 👤 User
- [finding]
- [finding]

### 💾 Data
- [finding]
- [finding]

[... repeat for all 8 lenses]

## 💡 Key Discoveries
1. **[Surprising finding]** — [why it matters]
2. **[Hidden requirement]** — [why it matters]
3. **[Blind spot]** — [why it matters]

## Prioritized Recommendations
1. [Most important action]
2. [Second most important]
3. [Third]
```
