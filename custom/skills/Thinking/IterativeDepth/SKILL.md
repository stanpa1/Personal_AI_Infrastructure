# IterativeDepth — 8-Lens Deep Analysis

Systematically examine a problem through 8 different lenses to uncover hidden requirements, blind spots, and non-obvious implications.

## The 8 Lenses
See `TheLenses.md` for detailed lens descriptions.

1. 👤 **User** — Who uses this and how?
2. 💾 **Data** — What data flows where?
3. ⚡ **Performance** — Speed, resources, limits?
4. 🔒 **Security** — Attack surface, permissions?
5. 🔄 **Lifecycle** — Deploy, maintain, deprecate?
6. 🤝 **Integration** — Dependencies, contracts?
7. 📊 **Observability** — Monitor, debug, audit?
8. 🌍 **Context** — Business, legal, cultural?
