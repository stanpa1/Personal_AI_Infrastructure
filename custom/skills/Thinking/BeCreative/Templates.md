# Idea Generation Templates

## SCAMPER Method
- **S**ubstitute — What can be replaced?
- **C**ombine — What can be merged?
- **A**dapt — What can be borrowed from elsewhere?
- **M**odify — What can be changed, enlarged, shrunk?
- **P**ut to other use — What else could this do?
- **E**liminate — What can be removed?
- **R**everse — What if we flip it?

## Random Input
Pick a random word/concept and force connections to the problem.

## Analogy Mining
"This problem is like [X] because..."
Try: nature, music, cooking, sports, architecture, games

## Worst Possible Idea
Generate the worst solutions deliberately. Then flip them.

## Time Travel
- How would this be solved in 1900?
- How would this be solved in 2100?
- How is it solved in a parallel universe?
