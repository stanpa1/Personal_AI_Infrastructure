# Scientific Templates

## Hypothesis Template
```
**H[n]:** [Statement of what you believe is true]
**Prediction:** If H[n] is true, then [observable outcome]
**Test:** [How to verify the prediction]
**Result:** [Confirmed / Refuted / Inconclusive]
```

## Experiment Template
```
**Experiment:** [Name]
**Goal:** Test H[n]
**Setup:** [What to configure]
**Steps:**
1. [step]
2. [step]
**Expected result:** [if hypothesis is correct]
**Actual result:** [what happened]
**Conclusion:** [what this tells us]
```
