# FullCycle Workflow

Complete scientific method cycle for thorough investigation.

## Trigger
User says "full scientific analysis", "systematic investigation", "rigorous analysis"

## Steps

1. **Literature review** — What's already known? Check existing docs, logs, code.
2. **Observation phase** — Spend time gathering data before hypothesizing
3. **Hypothesis generation** — Multiple competing explanations
4. **Experimental design** — Controlled experiments with clear success criteria
5. **Execution** — Run experiments, document everything
6. **Analysis** — Statistical or logical analysis of results
7. **Peer review** — Present findings, invite challenges
8. **Publication** — Document conclusions for future reference

## Output Format

```
# 🔬 Scientific Analysis: [Topic]

## 1. Prior Knowledge
[What we already know]

## 2. Observations
[Raw data and observations]

## 3. Hypotheses
[Numbered, falsifiable hypotheses]

## 4. Experimental Design
[Controlled experiments for each hypothesis]

## 5. Results
[Data from experiments]

## 6. Analysis
[What the data tells us]

## 7. Conclusions
**Primary finding:** [main conclusion]
**Secondary findings:** [additional insights]
**Limitations:** [what this analysis doesn't cover]
**Future work:** [recommended follow-up]

## 8. Record
[Summary for future reference]
```
