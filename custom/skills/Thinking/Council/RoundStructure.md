# Round Structure

## Round 1: Opening Positions
Each agent presents their position on the problem:
- State their recommendation
- Key arguments (2-3 points)
- Primary concerns with other approaches

**Format per agent:**
```
### [Emoji] [Role]: [1-line position]
**Recommendation:** [specific recommendation]
**Arguments:**
1. [point]
2. [point]
**Concern:** [what worries them about alternatives]
```

## Round 2: Challenge & Defend
Each agent responds to the strongest counter-argument:
- Acknowledge valid criticisms
- Defend or modify their position
- Identify the crux disagreement

**Format per agent:**
```
### [Emoji] [Role] responds:
**Acknowledges:** [valid point from another agent]
**Defends:** [why their position still holds]
**Modifies:** [any position updates]
**Crux:** [the fundamental disagreement]
```

## Round 3: Synthesis
Combine insights into a unified recommendation:
- What all agents agree on
- Resolved disagreements (with reasoning)
- Remaining risks and mitigations
- Final recommendation with confidence level

**Format:**
```
## 🤝 Council Synthesis
**Consensus:** [what everyone agrees on]
**Resolved:** [disagreements settled and how]
**Risks:** [acknowledged remaining risks]
**Recommendation:** [final unified recommendation]
**Confidence:** [High/Medium/Low] — [why]
```
