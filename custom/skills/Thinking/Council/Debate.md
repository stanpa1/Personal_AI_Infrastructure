# Debate Workflow

Full 3-round structured debate.

## Trigger
User says "council", "debate this", "discuss from multiple perspectives"

## Steps

1. **Frame the question** — restate the problem as a clear decision
2. **Round 1** — each agent gives opening position (see RoundStructure.md)
3. **Round 2** — each agent challenges and defends (see RoundStructure.md)
4. **Round 3** — synthesize into recommendation (see RoundStructure.md)
5. **Output** — format per OutputFormat.md

## Guidelines
- Each agent should genuinely advocate for their perspective
- Don't create false consensus too early
- The synthesis should acknowledge real trade-offs
- Confidence level should reflect actual agreement strength
