# Reconstruct Workflow

Full first-principles cycle: Deconstruct → Challenge → Reconstruct.

## Steps

1. Run **Deconstruct** workflow
2. Run **Challenge** workflow
3. **Reconstruct** — Build solution using only proven fundamentals
4. **Compare** — How does the new solution differ from the original?
5. **Decide** — Is the first-principles solution better? Why or why not?
